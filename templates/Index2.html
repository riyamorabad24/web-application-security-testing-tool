About WebScan

WebScan is an open-source web application vulnerability scanner built with Python and Flask. It automates detection of common security weaknesses aligned with the OWASP Top 10.

OWASP Top 10 Coverage

✓ SQL Injection
✓ Broken Access Control
✓ XSS (Cross-Site Scripting)
✓ Security Misconfigurations
✓ CSRF Protection
✓ Security Headers
✓ SSL / TLS Issues
✓ Sensitive Endpoint Exposure

Legal Notice

⚠ Authorized use only. This tool is for educational and security testing on systems you own or have explicit written permission to test. Unauthorized scanning may violate computer fraud laws. The developers assume no liability for misuse.
